<b>[Documentation](https://www.resolid.tech/docs/log)</b> | [Framework Bundle](https://github.com/resolid/framework)

## Installation

```shell
pnpm add @resolid/app-log-file
# or
npm install @resolid/app-log-file
# or
yarn add @resolid/app-log-file
# or
bun add @resolid/app-log-file
```
